# Sistema de Gestión de Lista de Tareas

## Descripción

Aplicación de consola desarrollada mediante Programación Orientada a Objetos que permite administrar una lista de tareas utilizando un ArrayList. El sistema facilita el registro, seguimiento y control de actividades pendientes y completadas. :contentReference[oaicite:0]{index=0}

## Funcionalidades

### Agregar tarea
Permite registrar una nueva tarea indicando su nombre y descripción.

### Marcar tarea como completada
Permite cambiar el estado de una tarea a completada mediante su número en la lista.

### Gestionar tareas
Cada tarea almacena información relacionada con su nombre, descripción y estado de realización.

### Visualizar tareas
Muestra todas las tareas registradas, diferenciando las pendientes de las completadas mediante un indicador visual.

### Salir
Finaliza la ejecución del programa.

## Conceptos Aplicados

- Programación Orientada a Objetos
- Encapsulamiento
- Constructores
- Getters y Setters
- ArrayList
- Gestión de objetos
- Menús interactivos por consola
